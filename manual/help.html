ส่วนของสินค้าคงคลัง (Inventory)

Item management

1. Item List (ทะเบียนสินค้า)
1.1  สามารถค้นหาสินค้าที่ต้องการจะดูรายละเอียด ได้จากการระบุ
Item Name (ชื่อสินค้า) โดยพิมพ์แค่บางส่วนของชื่อหรือทั้งหมดก็ได้
Brand ต้องพิมพ์ชื่อยี่ห้อให้ถูกต้อง
ชื่อSupplier คลิ๊กแล้วเลือกตามรายชื่อที่มีให้
Item ID ต้องระบุให้ถูกต้อง
Barcode สินค้าเก่าจะกำหนด Barcode เป็นเลขเดียวกับ Item Id ถ้าเพิ่มสินค้าใหม่ให้กำหนดเลขที่เจ๊กอินต้องการ ตอนที่เพิ่มสินค้า
	เสร็จแล้วกดปุ่ม Search
	ถ้าต้องการลบข้อมูลที่พิมพ์ไปทั้งหมด กดปุ่ม Reset
1.2 ถ้าต้องการเปลี่ยนแปลงรายละเอียดของสินค้า ให้คลิ๊กที่ Item ID ของสินค้านั้น
โปรแกรมจะเปลี่ยนเป็นหน้าที่แสดงรายละเอียดของสินค้าเหมือนตอนที่เพิ่มสินค้าใหม่ ให้แก้ไขรายละเอียดที่ต้องการ แล้วกดปุ่ม Update

	ถ้าต้องการแก้ไขUnit Price (ราคาขาย) ให้ใช้ฟังก์ชั่น Set Priceที่อยู่ภายใต้แท๊บ Item Management นี้เช่นกัน
	ถ้าต้องการแก้ไขInitial stock (จำนวนสินค้าที่ซื้อมาล็อตแรก)  ให้ใช้ฟังก์ชั่น Adjust Inventory ที่อยู่ภายใต้แท๊บ Stock Management

2. New Item (เพิ่มสินค้าใหม่)
พิมพ์รายละเอียดของสินค้าใหม่ดังนี้
Item Name คือ ชื่อสินค้าเป็นภาษาไทย
Item Name in English ถ้าไม่ได้ใช้ ไม่ต้องใส่ก็ได้
Brand คือ ชื่อยี่ห้อ จะใส่เป็นภาษาไทย หรือ อังกฤษก็ได้
Product code คือ รหัสใช้เรียกแทนสินค้า
Supplier เลือกSupplierที่ซื้อสินค้ามา (ถ้าเป็นSupplierใหม่ ต้องเพิ่มSupplierก่อน)
Color ถ้าไม่ต้องการระบุสี ไม่ต้องใส่ก็ได้
Unit Cost คือ ต้นทุนของสินค้าทั้งหมด
Unit Price คือ ราคาขายของสินค้า
Initial Stock คือ จำนวนสินค้าที่ซื้อมาล็อตแรก
Barcode ของเจ๊กอิน ให้กำหนดเลขที่ตอ้งการ แต่ต้องไม่ซ้ำกับสินค้าตัวอื่น ถ้าจะใช้ Item Idเป็นBarcode เหมือนสินค้าเก่า ก็ไม่ต้องใส่ค่ะ เพราะItem IDโปรแกรมจะกำหนดให้อัตโนมัติ
	เมื่อตรวจสอบรายละเอียดทั้งหมดว่าถูกต้องแล้ว กดปุ่ม Submit
	ถ้าต้องการลบข้อมูลที่พิมพ์ไปทั้งหมด กดปุ่ม Reset
